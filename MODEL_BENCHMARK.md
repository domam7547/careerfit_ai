# MODEL_BENCHMARK.md
# 나만의 AI 모델 벤치마크 문서

## 1. 문서 목적

이 문서는 CareerFit AI 프로젝트에서 사용할 AI 모델을 선택하기 위해 작성한다.

공개 리더보드 점수만 보고 모델을 고르지 않고,
내 프로젝트의 실제 사용 목적에 맞게 직접 질문하고 비교한다.

CareerFit AI의 모델 선택 기준은 다음과 같다.

* 한국어 답변 품질
* 취업·직무 상담 적합성
* 채용공고 요약 능력
* 사용자 맥락 반영 능력
* 환각이 적은 답변
* 응답 속도
* 비용 또는 무료 사용 가능성
* API 연결 난이도
* 라이선스와 사용 제한

## 2. 평가 대상 모델

| 번호 | Provider | Model | 용도 | 상태 |
| -- | ------------ | --------------------- | ------------- | ------ |
| 1 | Gemini | Gemini 2.5 Flash-Lite | 기본 LLM 후보 | 한도 초과로 실측 보류 |
| 2 | Mistral | mistral-small-latest | 무료 API 비교 후보 | 1차 평가 완료 |
| 3 | Hugging Face | Qwen/Qwen2.5-0.5B-Instruct | 무료/저비용 LLM 후보 | 1차 평가 완료 |
| 4 | Ollama | llama3.2:3b | 오프라인/로컬 실습 후보 | 1차 평가 완료 |
| 5 | Mock | Mock Response | 한도 초과 대비용 | 1차 평가 완료 |

> 실제 모델명은 테스트 시점에 사용 가능한 모델로 업데이트한다.

### 2-1. 1차 평가 대상 선정 메모

* Gemini 2.5 Flash-Lite
  * 현재 프로젝트 코드와 가장 바로 연결되므로 기본 LLM 후보로 유지한다.
* mistral-small-latest
  * 무료 모드에서 API 테스트가 가능해 Gemini 대체 비교 후보로 선정한다.
  * 현재 `llm_service.py`에 provider 분기 코드가 이미 있어 연결 난이도가 낮다.
* Qwen/Qwen2.5-0.5B-Instruct
  * Hugging Face 비교 후보로 선정한다.
  * Apache-2.0 라이선스와 소형 모델 크기로 빠른 비교 테스트에 적합하다.
* llama3.2:3b
  * Ollama 로컬 실습 후보로 선정한다.
  * 로컬 실행과 비용 통제 측면에서 유리하다.
* Mock Response
  * API 한도 초과 또는 실습 환경 이슈 발생 시 fallback 용도로 유지한다.

## 4. 평가 방식

모델 평가는 1차 정량 평가와 2차 정성 평가로 나눈다.

### 4-1. 정량 평가

각 항목을 1점부터 5점까지 평가한다.

| 점수 | 의미 |
| -- | -------- |
| 5점 | 매우 좋음 |
| 4점 | 좋음 |
| 3점 | 보통 |
| 2점 | 부족함 |
| 1점 | 사용하기 어려움 |

### 4-2. 정성 평가

점수만으로 판단하지 않고, 실제 답변을 읽고 아래 내용을 기록한다.

* 답변이 자연스러운가?
* 사용자의 전공, 경험, 목표를 반영했는가?
* 없는 정보를 지어내지 않았는가?
* 너무 일반적인 조언만 하지 않았는가?
* 실제 포트폴리오 개선에 도움이 되는가?
* 답변 형식이 보기 쉬운가?
* 한국어 표현이 어색하지 않은가?

## 10. 최종 선택 기록

### 최종 선택 모델

| 용도 | 선택 모델 | 선택 이유 |
| -------- | ------------- | ------------------------- |
| 기본 LLM | mistral-small-latest (잠정) | 2026-07-06 1차 테스트 기준으로 한국어 품질, 구조화, 직무 적합성이 가장 안정적이었다. Gemini는 기준 후보로 유지하되, 한도 초과로 실측 비교가 보류되어 현재는 Mistral을 우선 사용하기 적절하다. |
| 임베딩 모델 | 미정 | 이 문서는 LLM 중심 벤치마크 문서이며, 임베딩 모델 비교 기록은 아직 별도로 진행하지 않았다. |
| 무료 실습용 | Mistral free mode / Ollama llama3.2:3b | Mistral은 무료 모드에서 API 테스트가 가능하고 한국어 품질이 더 좋다. Ollama는 비용 없이 계속 실습할 수 있지만 답변 품질은 보조 용도에 가깝다. |
| 한도 초과 대비 | Mock Response | API 한도나 비용 문제 없이 수업 진행 가능 |

Gemini는 한국어 답변 품질과 직무 상담 문맥 이해가 좋아 기본 LLM으로 사용한다.
